[ DashboardPage.jsx ]

로그인하면 처음 보이는 화면
지금까지 만들어진 코드 리뷰 세션들을 카드 형태로 보여주고,
새 세션을 만들 수 있는 버튼과 모달을 제공



[ 핵심 용어 설명 ]

▶ useEffect + 빈 의존성 배열 []
   - 컴포넌트가 처음 화면에 나타날 때(마운트) 딱 한 번 실행
   - 세션 목록을 처음 한 번 불러오기 위해 사용

▶ 모달 (Modal)
   - 화면 위에 뜨는 팝업 창
   - 배경을 반투명하게 가리고 중앙에 내용을 표시
   - showModal state로 보이기/숨기기 제어

▶ Authorization 헤더
   - 모든 API 요청에 JWT 토큰을 붙여서 보내는 방법
   - headers: { Authorization: `Bearer ${token}` }
   - 백엔드 authMiddleware가 이 헤더를 읽어서 인증 처리

▶ prev (이전 상태)
   - setSessions(prev => [data, ...prev])
   - React state 업데이트 시 이전 값을 기반으로 새 값 계산
   - ...prev = 기존 배열을 펼침 (스프레드 연산자)
   - 새 세션을 맨 앞에 추가하는 방법
